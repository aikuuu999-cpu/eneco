## Брейкпоинты

### Основные брейкпоинты:
- **Desktop**: > 768px
- **Tablet**: ≤ 768px
- **Mobile**: ≤ 480px
- **Small Mobile**: ≤ 320px

### Дополнительные брейкпоинты:
- **600px**: Переход сетки каталогов к одной колонке
- **640px**: Оптимизация типографики для планшетов
