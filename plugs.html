Фасонные части для герметичного закрытия трубопроводов. Заглушки из литого чугуна для соединения с трубами SML обеспечивают надежное закрытие концов трубопроводных систем.

ЧУГУННЫЕ ЗАГЛУШКИ SML

Чугунные заглушки SML – качественные фасонные части для герметичного закрытия трубопроводов. Произведены из высококачественного литого чугуна с соблюдением всех технических стандартов DIN EN 877. Обеспечивают надежное соединение с трубами SML и долговечную эксплуатацию в системах канализации. Представлены в двух типах: заглушки с прижимными скобами для временного закрытия и пресс-заглушки для постоянного герметичного закрытия системы. Максимальное допустимое давление – 0,2-0,5 бар.
